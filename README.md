# Лендинг · ИИ-агенты для Страна Девелопмент

Прототип одностраничного лендинга в стиле Aurus Residences (премиум + анимации по скроллу).
Стек: чистый **HTML / CSS / JS + GSAP ScrollTrigger** (без сборки — открывается напрямую).

## Структура

```
index.html        # разметка, 4 блока
css/styles.css    # дизайн-система (палитра/шрифты — в :root сверху)
js/main.js        # анимации (GSAP + ScrollTrigger + SmoothScroll)
```

## Запуск

Просто открыть `index.html` в браузере (или `file://`-путь). Интернет нужен для CDN-библиотек и шрифтов.

## 5 блоков (наполнены данными из СС ИИ 2026)

1. **Титул / Hero** — «Применение ИИ в компании», reveal текста, parallax, мета-строка (формат/место/для кого).
2. **Расписание** — вертикальный таймлайн дня (10:00 → 17:50), стаггер-появление строк.
3. **Почему сейчас** — закреплённая секция, фон фирменный градиент → графит; слоган «Кризис — это окно возможностей», хук про G7.
4. **Эффективность** — риски бездействия + сигнатурная формула **КДТ** (Культура × Данные × Технологии), «Множитель ноль — результат ноль».
5. **Тренинг для ТОПов** — программа, воркшоп «Сквозной учёт. Драйверы», форма записи + футер.

### Открытые вопросы по контенту
- **Дата сессии**: на титульном слайде 14.10.2025, на слайде деталей 23.07.2026 — на лендинге дата пока не выводится, нужно подтвердить.
- **Тайминг расписания** ориентировочный (на слайдах две версии) — уточнить финальную раскадровку.
- **Шрифт бренда** (Strana Font / Circe) не подключён — стоит Unbounded + Manrope.

## Куда вставлять контент

Все места под наполнение помечены `{{ ... }}` прямо в `index.html`. Основные:

| Плейсхолдер | Где | Что вставить |
|---|---|---|
| `{{HERO_EYEBROW}}` | Hero | мелкая надпись над заголовком |
| `{{HERO_LEAD}}` | Hero | 1–2 предложения о сути |
| `{{CTA_PRIMARY}}` | Hero | текст главной кнопки |
| `{{STATS_TITLE}}` | Stats | крупное утверждение об эффекте |
| `data-count-to="0"` | Stats | **целевые числа метрик** (напр. `data-count-to="96"`) |
| `{{ед.}}` / `{{подпись метрики}}` | Stats | единица и подпись под цифрой |
| `{{FEATURES_TITLE}}` | Features | заголовок блока возможностей |
| `{{CARD_N_TITLE/TEXT}}` | Features | 4 карточки (заголовок + описание) |
| `{{CTA_TITLE/LEAD/BUTTON/NOTE}}` | CTA | финальный призыв и форма |

### Цифры в Stats
Анимация count-up берёт целевое значение из атрибута `data-count-to`.
Для дробей добавь `data-decimals="1"`. Пример:
```html
<span class="stat__num" data-count-to="225.7" data-decimals="1">0</span>
```

## Правка дизайна

Цвета и шрифты — переменные в начале `css/styles.css` (блок `:root`).
Палитра взята из официального брендбука: **brandbook.strana.com**.
- `--accent` `#C7017F` — маджента (ядро фирменного градиента)
- `--accent-ink` `#951B81` — черника (тёмный ховер)
- `--accent-red` `#E30613` — красный (конец градиента)
- `--grad-brand` — фирменный градиент пурпур→маджента→красный (лого, фон Stats, свечение hero)
- `--ink` `#29303E` — графит (фирменный тёмный)
- `--paper` `#F4F1EC` — тёплый светлый фон
- `--font-display` / `--font-text` — Unbounded / Manrope

> Примечание по шрифту: у бренда кастомный «Strana Font» + Circe. Их нет в открытом доступе
> (CDN), поэтому пока стоит близкий по духу Unbounded. Если есть файлы Strana Font / Circe —
> подключим через @font-face и заменим один-в-один.

## Заметки

- Библиотеки те же, что у эталона Aurus: GSAP 3, ScrollTrigger, SmoothScroll.
