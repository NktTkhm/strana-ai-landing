03 · Вводная часть

Слайд 01 · Карта понятий

Из чего состоит искусственный интеллект

Договоримся о терминах: внизу — данные и модели, наверху — системы, которые сами доводят задачу до результата.

  • Фундамент. Data Science и ML извлекают закономерности из данных — без этого верхние уровни не работают.
  • Проявления. LLM и генеративный ИИ создают контент, но сами не ведут бизнес-процесс.
  • Действие. Агент = модель + память + инструменты + цель. Здесь ИИ входит в процессы Страны.

Слайд 02 · Внедрение

Путь к эффективному внедрению ИИ

Три барьера по порядку: технологии → данные → культура. Пропустить этап нельзя — только последовательное преодоление.

  • 1. Технологии. Инфраструктура, серверы, модели ИИ и API — фундамент, без которого дальше не поехать.
  • 2. Данные. Доверие и качество, контекст и семантика, доступность данных — второй барьер на пути.
  • 3. Культура. Люди, навыки, скорость и привычка к ИИ — финальный рубеж перед эффективным внедрением.
